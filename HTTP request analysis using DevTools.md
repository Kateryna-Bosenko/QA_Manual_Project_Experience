## Conducted a thorough analysis of HTTP requests and responses while interacting with an e-commerce website.
 ##### Task 1:
1. Open a browser (e.g. Chrome) and DevTools (Ctrl+Shift+I).
2. Enter the link https://www.makeup.com.ua/ in the address bar.
3. Press Enter.
4. In DevTools, go to the Network tab.
5. Find the query "www.makeup.com.ua" in the Name column
6. Fill in the "Task #1" sheet of Google Sheets for this query.
 ##### Task 2:
1. Repeat steps 1-5 from Task 1 for the query https://makeup.com.ua/
2. Fill in the "Task #2" sheet of Google Sheets for this query.
 ##### Task 3:
1. Without closing DevTools, add any product to the cart on the current page of the site.
2. In DevTools, analyze the query "cart/", its method and status. For convenience, you can use the filter as shown in the picture below, and write the word "cart/" in the filter field.
3. Fill in the "Task №3" sheet of Google Sheets for this query.
 ##### Task 4:
1. Removed the product from the cart.
2. Another "cart/" query will appear in DevTools. Analyze its method and status.
3. Fill in the "Task №4" sheet of Google Sheets for this query.
 ##### Task 5:
1. Removed the word “cart/” in the filter field (if it is entered).
2. Repeat steps 1-5 from Task 1 for the query https://www.makeup.ua/
3. Fill in the "Task №5" sheet of Google Sheets for this query.
 ##### Task 6:
1. Repeat steps 1-5 from Task 1 for the query https://makeup.com.ua/ua/product/00000/
2. In DevTools, find the query "00000/". Analyze its method and status.
3. Fill in the "Task #6" sheet of Google Sheets for this query.
 ##### Task 7:
1. Repeat steps 1-5 from Task 1 for the query https://graph.makeup.com.ua/chat/user/me
2. In DevTools, find the query "me". Analyze its method and status.
3. Fill in the "Task #7" sheet of Google Sheets for this query.
